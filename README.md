# Smart-Campus-Timetabling-Resource-Allocation

The **Smart Campus Timetabling System** is a **constraint-aware academic scheduling platform** that generates **conflict-free timetables** using real-world institutional rules. Unlike static or manual systems, it uses an **automated scheduling engine** enforcing faculty availability, workload limits, room constraints, and batch-level rules.

The system is **role-based**, **secure**, and **transactional**, ensuring concise and correct timetable generation.

## Technology Stack

### Backend
- **Node.js**
- **Express**
- **Prisma ORM**
- **PostgreSQL / MySQL**
- **JWT-based authentication**
- **Role-based authorization**

### Frontend
- **React (Vite)**
- **React Router**
- **Plain HTML tables**

## Key Features
- **Automated timetable generation**
- **Faculty, room, and batch conflict prevention**
- **Faculty availability and workload enforcement**
- **Support for THEORY and LAB courses**
- **Role-based access for Admin, Faculty, and Students**
